# callVU

## Project setup
```
1.download or clone the project 
2.  Open Cmd Navigate into backend folder and run npn I 
3. run node server.js 
4. Navigate into frontend and run npm I
5. run npm run serve
6. create a folder to track in C:\
7. open browse and go to http://localhost:8080/
8. log in with "user@user" password "1234"
9. input name of folder you created in  C:\ in to the input field 
10. add or remove or edit folders or files and watch them appear in log history
```
